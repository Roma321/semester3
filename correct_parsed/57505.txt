API  1. Application Programming Interface  интерфейс прикладного программирования набор стандартных программных прерываний, вызовов процедур  и форматов данных, которые должны использовать прикладные программы для запроса и получения от операционной системы, телекоммуникационного протокола или программного интерфейса  связанного с ними обслуживания.
