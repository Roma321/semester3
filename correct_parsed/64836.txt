EXEфайлы для Windows и OS2 используют другие форматы для основной части программы, но всё равно начинаются с заглушки в формате MZ, которая, как правило, при попытке запустить файл в DOS выводит сообщение This program cannot be run in DOS mode.  и завершает выполнение, хотя теоретически может запускать некий произвольный код, работоспособный в DOS.
