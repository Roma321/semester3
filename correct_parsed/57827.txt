ASCII  American Standard Code for Information Interchange   американский стандартный код для обмена информацией, код ASCII набор из 128 кодов символов для машинного представления прописных и строчных букв латинского алфавита, чисел, знаков препинания и специальных символов, каждому из которых соответствует конкретное 7битовое двоичное число.
