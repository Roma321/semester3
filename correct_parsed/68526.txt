HSTR  HighSpeed Token Ring  высокоскоростная  Token Ring, спецификация HSTR разрабатывается рабочей группой IEEE 802.5 в виде трех спецификаций802.5t  описывает передачу данных по неэкранированной витой паре категории 5802.5u  описывает физический уровень при передаче по оптоволокну802.5v  описывает передачу данных с гигабитными скоростями .Англорусский словарь компьютерных терминов
