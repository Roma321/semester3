Memory Stick SelectТак как объем первоначальных Memory Stick был ограничен 128 Мбайтами, некоторое время выпускались специальные карточки Memory Stick Select, в которых находилось 2  чипа по 128 Мб каждый, причем устройство пользовалось только одним из них, механический переключатель находился на самой карте.
