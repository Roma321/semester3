TAPI  Telephony API  API для телефонии, интерфейс программирования приложений телефонной связи, интерфейс TAPI набор интерфейсов в WOSA, используемый клиентскими приложениями Windows для аппаратнонезависимого взаимодействия с различными типами коммуникационного оборудования, в частности, модемами и факсимильными аппаратами, а также для программ компьютерной телефонии.
