Java Virtual Machine   виртуальная машина Java  основная часть исполняющей системы Java, так называемой Java Runtime Environment . Виртуальная машина Java исполняет байткод Java, предварительно созданный из исходного текста Javaпрограммы компилятором Java . JVM может также использоваться для выполнения программ, написанных на других языках программирования.
