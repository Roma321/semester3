Java Cryptography Extension___6675

Криптографическое расширение Java   официально выпущенное стандартное расширение для платформы Java и часть Java Cryptography Architecture . Представляет собой набор пакетов, который обеспечивает фреймворк и реализацию таких криптографических задач, как шифрование и расшифрование данных, генерация и проверка на подлинность ключей управления, а также реализацию алгоритмов Message Authentication Code .
