SQL Server 2008 R2 SP1 содержал исправления ошибок, на которые Microsoft получила жалобы от клиентов через службу Windows Error Reporting, а также некоторые улучшения функциональности динамические административные представления , повышение скорости выполнения запросов при помощи технологии ForceSeek, технология Datatier Application Component Framework  для упрощения обновления БД, контроль доступного места на жёстком диске для PowerPivot.
